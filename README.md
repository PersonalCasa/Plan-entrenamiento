# 🏁 Medio Maratón Punta del Este — Plan 16 semanas

App personal de entrenamiento para el Medio Maratón de Punta del Este (6 Sep 2026).

## Funcionalidades

- Plan de 16 semanas: Base → Construcción I → Construcción II → Taper
- Días: Martes · Jueves · Domingo (alternativa Lun · Mié · Vie)
- Semanas pasadas se colapsan automáticamente en "Historial"
- Semana actual destacada con badge ▸ HOY
- Tick por sesión: cumplida ✓ / no cumplida ✗ con nota de motivo
- Estado guardado en localStorage (persiste entre sesiones)
- Pestaña Diccionario: zonas Z1–Z4, tipos de sesión y ejercicios de gimnasio

## Stack

HTML + CSS + JS vanilla · GitHub Pages · localStorage

## Deploy

Activar GitHub Pages en Settings → Pages → Branch: main → / (root)
